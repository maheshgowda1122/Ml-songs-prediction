# 🎵 Song Popularity Prediction using Machine Learning

## 📌 Project Overview
This project explores a Spotify tracks dataset to analyze and predict song popularity using machine learning techniques using Jupyter Notebooks.

## 📂 Dataset
- File: tracks.csv
- Target Variable: popularity
- Features: energy, tempo, valence, danceability, speechiness, liveness

## 📓 Notebooks
- RQ_1_assignment_ML.ipynb → Data preprocessing & feature importance
- RQ_2_ML_assignment.ipynb → Model comparison (Regression)
- RQ_3.ipynb → Emotional features analysis
- RQ_4.ipynb → Classification (Hit prediction)
- RQ_5.ipynb → Additional analysis

## ⚙️ Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 🚀 How to Run

1. Install dependencies:
pip install pandas numpy matplotlib seaborn scikit-learn

2. Run Jupyter Notebook:
jupyter notebook

3. Open notebooks and run in order:
RQ1 → RQ2 → RQ3 → RQ4 → RQ5

## 📁 Outputs
- RQ2_model_results.csv
- RQ3_emotional.pdf
- RQ4_roc.pdf
- RQ6_clusters.pdf

## 📈 Key Insights
- Predicting popularity is complex
- Emotional features alone are weak predictors
- Random Forest performs better than Linear Regression
